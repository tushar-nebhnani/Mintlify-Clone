# Mintlify Landing Page Clone

A pixel-perfect, high-fidelity recreation of the [Mintlify](https://mintlify.com/) landing page. This project demonstrates modern front-end development techniques using semantic HTML5 and advanced CSS3, focusing on responsiveness, glassmorphism, and precise layout structures.

## âœ¨ Key Features

* **Glassmorphism Effects:** Custom `backdrop-filter` and semi-transparent borders used in the Hero email input and navigation elements.
* **Responsive Layouts:**
    * **CSS Grid:** Used for the "Customer Stories" and "Footer Links" sections to handle complex multi-column layouts.
    * **Flexbox:** Used for the Navbar, Hero alignment, and feature lists to ensure perfect centering and spacing.
* **Pure CSS Styling:** No external frameworks (like Bootstrap or Tailwind) were used. All gradients, shadows, and transitions are hand-coded.
* **SVG Integration:** All icons (social media, UI elements, logos) are embedded directly as SVGs for fast loading and crisp scaling.
* **Interactive Elements:**
    * Hover effects on customer cards (lift + shadow).
    * Focus states on input fields (glow effect).
    * Animated buttons.
